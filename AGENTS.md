# FamilyLedger Agent Instructions

This project is a family financial management web app.

Main principles:
- Keep the UI clean, simple, and family-friendly.
- Use TypeScript strictly.
- Use reusable components.
- Do not connect to real bank APIs.
- Store data in Supabase with server-side data access.
- Format money in Indonesian Rupiah.
- Prefer clear code over clever code.
- Make every page responsive.
- Do not leave placeholder buttons that do nothing unless clearly marked as future work.

Before finishing any task:
- Run build checks.
- Remove unused imports.
- Check mobile layout.
- Update README if the feature changes user-facing behavior.
